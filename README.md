
# Useful scripts because my dumbass can't remember them

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

The page will reload when you make changes except when it doesn't because of tailwind.

### `npm run build`

Builds the app for production to the `build` folder.\
It correctly bundles React in production mode and optimizes the build for the best performance.

### `npx tailwindcss -i ./src/assets/css/input.css -o ./src/assets/css/output.css --watch`

Makes it so that any changes will update output.css

### `npm run deploy`
Runs "npm run build" and also deploys it to github pages.
